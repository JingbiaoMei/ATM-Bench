# OpenClaw Docs

OpenClaw has a separate benchmark-user workflow and a separate threat model
from the other runners, so its docs live here.

Start here:

- `agent_systems/openclaw/README.md`

Then read in this order:

1. `agent_systems/openclaw/benchmark_quickstart.md`
2. `agent_systems/openclaw/isolation_setup.md`

File roles:
- `README.md`: top-level index for the OpenClaw benchmark docs
- `benchmark_quickstart.md`: canonical operator runbook
- `isolation_setup.md`: rationale, threat model, and audit guidance
